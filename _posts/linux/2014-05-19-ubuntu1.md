##ubuntu 基本命令一：

	1、帮助
    -help简单的帮助
    help command 较详细的帮助
    man command 最详细的帮助
	2、ls 命令
	      ls -a 显示全部的文件及文件夹，包括隐藏的文件或文件夹
	      ls -l 显示较全的文件信息，包括权限、用户、用户组等。
	      ls --color 显示文件及文件夹，并标有不同的颜色。
	3、tab键
	      tab command 用于当你的命令记不全时，输入一部再按一下进行补全，如果有多个前面部分相同命令，则
	      按两次tab键
	4、alias
	      alias ubuntu="ls" 用于为一个命令取别名。当你输入ubuntu时等价于输入了ls命令。
